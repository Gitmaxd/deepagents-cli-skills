# Contributing

This repository is a catalog/index for the deepagents CLI Agent Skill. It does not contain skill code.

To contribute to a specific skill variant, visit the appropriate repository:

- [deepagents-cli-deepagents-skill](https://github.com/Gitmaxd/deepagents-cli-deepagents-skill)
- [deepagents-cli-claude-skill](https://github.com/Gitmaxd/deepagents-cli-claude-skill)
- [deepagents-cli-codex-skill](https://github.com/Gitmaxd/deepagents-cli-codex-skill)
- [deepagents-cli-droid-skill](https://github.com/Gitmaxd/deepagents-cli-droid-skill)

For issues with this catalog page (broken links, outdated information), please [open an issue](https://github.com/Gitmaxd/deepagents-cli-skills/issues).
